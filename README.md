# Day Aurelia Modal
### Current Version: 1.7.0

Day-Aurelia-Modal - aurelia-framework modal package.

## Installing

```
npm install day-aurelia-modal

yarn add day-aurelia-modal
```

## Docs & Demo (1.7.0)
